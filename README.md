# 스꾸친 (SKKUCHIN)
스꾸친은 성균관대학교 학생들 간의 밥 약속을 매칭시켜주는 서비스입니다.

<br>

## 사용 기술
- Front-End: React.js, Redux, Apache Cordova
- Back-End: Django
- Server: Nginx
- Cloud: AWS (EC2, S3, RDS), Firebase
- Database: MySQL
